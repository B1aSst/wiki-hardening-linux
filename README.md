# wiki-hardening-linux

Dans le cadre du cours Durcissement OS de 2ème année à l'ENSIBS. Voici le wiki de durcissement pour système Linux.

Il se base en priorité sur les [guides de l'ANSSI](https://cyber.gouv.fr/publications).

Le wiki est disponible à l'adresse suivante : [wiki hardening linux](https://b1asst.github.io/wiki-hardening-linux/)

## Visualiser en local

Installer mkdocs dans votre environnement python virtuel préféré :
```sh
pip install mkdocs
```

Pour exposer le site en local :
```sh
mkdocs serve
```
